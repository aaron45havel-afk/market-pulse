Available template variables
Always available: {first_name}, {name}, {title}, {agency}, {my_name}.

Pulled from a saved discovery call (empty for contacts who haven't had one):
  • {call_summary} — Stage-2 executive summary
  • {pain_analysis} — Stage-2 ranked pain analysis
  • {mvp_scope} — Stage-2 MVP scope markdown
  • {win_condition} — the prospect's "if you got one thing right, what?" answer
  • {root_problem} — root problem description from the extraction
  • {problem_quote} — verbatim quote of the root problem (great for opening lines)
  • {concrete_story} — the "last time it went wrong" paragraph
  • {process_name} — name of the process they walked through
  • {time_cost} — first metric, formatted (e.g. "1 week")
